# Flutter 待办事项应用 - 新手教程

## 第一章：前言与概述

### 欢迎！

你好！欢迎来到这份 Flutter 新手教程。本教程将带你从零开始，一步步构建一个功能完整的待办事项（Todo）应用。

**即使你完全不懂 Flutter，甚至没有任何编程经验**，只要跟着教程走，你也能完成这个项目！

### 我们将构建什么？

一个现代化的待办事项应用，具备以下功能：

| 功能 | 描述 |
|------|------|
| ✅ 添加任务 | 快速添加新任务，支持设置截止日期 |
| ✅ 标记完成 | 点击复选框切换任务完成状态 |
| ✅ 编辑任务 | 修改已有任务的内容 |
| ✅ 删除任务 | 左滑或点击删除按钮删除任务 |
| ✅ 筛选任务 | 支持筛选"全部"/"进行中"/"已完成" |
| ✅ 本地存储 | 数据保存在本地，关闭应用不丢失 |
| ✅ 深色模式 | 自动跟随系统主题切换 |

### 技术栈介绍

本项目使用的技术：

- **Flutter** - Google 开发的跨平台 UI 框架，一套代码可以同时运行在 Android、iOS、Web、桌面
- **Dart** - Flutter 使用的编程语言，语法类似 JavaScript/Java
- **Provider** - 状态管理库，让数据在应用各组件间共享
- **SharedPreferences** - 本地存储，用于保存任务数据
- **Material Design 3** - Google 最新的设计语言，界面现代美观

### 学习路径

本教程分为以下几个章节：

1. **前言与概述**（本章）- 了解项目概况
2. **环境搭建** - 安装 Flutter 开发环境
3. **Flutter 核心概念** - 理解 Widget、State、Build 方法
4. **项目结构解析** - 了解代码组织方式
5. **数据模型详解** - Task 类的设计与实现
6. **状态管理详解** - Provider 的使用
7. **UI 组件详解** - 各个界面组件的实现
8. **关键技术深入** - Material Design 3、SharedPreferences
9. **常见问题排查** - 遇到问题怎么办
10. **扩展功能建议** - 如何继续改进应用

### 你需要准备什么？

- 一台电脑（Windows、macOS 或 Linux 都可以）

- 大约 2GB 的硬盘空间
- 学习的热情和耐心！
